# generate-vuetemp-kong
npm plugin, just for studying



一个快速生成页面文件的插件，搭配vue模版使用。  


## 项目结构


```javascript
.
├── bin           //命令配置
├── README.md     //说明文档
├── index.js      //主入口
├── src           //功能文件
├── package.json  //包信息

## Install 
  
  ```javascript
  $ npm i generate-vuetemp-kong -g 


```

## Useage

kong g [type] [name]     

name为 你需要生成的文件名
   
   
   目前支持的type：
   
   * page
   * component
